gptop is a cross-platform GPU monitor TUI. It provides real-time visibility into GPU utilization, memory, thermals, power draw, compute performance and per-process breakdowns. It currently supports Apple Silicon on macOS and NVIDIA GPUs on Linux, with more backends planned.
